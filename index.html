Introducción

Bienvenid@ al blog sobre lenguajes de programación. ¿Sabías que la programación no es un invento moderno, sino que sus raíces se remontan a siglos antes de la tecnología electrónica? Filósofos griegos y primeros ingenieros, como Herón de Alejandría(click aqui), ya crearon autómatas con mecanismos que podían seguir instrucciones específicas. Por ejemplo, Herón diseñó un carrito de madera que se movía y cambiaba de dirección gracias a un sistema de cuerdas y clavos que funcionaba como un lenguaje primitivo de programación.

La programación, tal como la conocemos hoy, está profundamente ligada a las matemáticas y al pensamiento lógico. Desde Ada Lovelace, considerada la primera programadora, hasta Alan Turing, quien formuló la teoría de la máquina que lleva su nombre, las bases científicas de la computación han sido fundamentales para el desarrollo de esta disciplina.

Aunque no es indispensable ser experto en matemáticas para aprender a programar, entender que los lenguajes formales de programación se basan en lógica matemática puede ayudarte a comprender mejor su funcionamiento. Existen miles de lenguajes de programación, cada uno con sus ventajas y usos.

En este blog exploraremos cuatro lenguajes populares: PHP, JavaScript, Python y Java. Te mostraré sus características principales y para qué tipo de proyectos son ideales. Espero que disfrutes este viaje por el mundo de la informática y la creatividad.

PHP

PHP (acrónimo recursivo de "PHP: Hypertext Preprocessor") es un lenguaje de programación del lado del servidor, especialmente diseñado para el desarrollo web. Fue creado por Rasmus Lerdorf en 1994 y se ha convertido en uno de los lenguajes más populares para construir sitios web dinámicos e interactivos. A diferencia de HTML, que solo estructura y presenta contenido estático, PHP permite interactuar con bases de datos, procesar formularios, gestionar sesiones y crear contenido dinámico. PHP es gratuito, de código abierto y está soportado por una gran comunidad de desarrolladores.

Declaracion de variables: Se inicializa con $ Ej: $musica="vallenato" $nombre="juan" 

        


<?php
// Definimos una variable con el nombre del usuario
$nombre = "Laura";

// Mostramos un saludo personalizado
echo "¡Hola, " . $nombre . "! Bienvenida al mundo de PHP.";
?>

Funciones de php

1. Procesar lógica del lado del servidor Ejecuta código en el servidor y envía el resultado al navegador en forma de HTML. Por ejemplo: mostrar datos personalizados, manejar formularios o verificar inicios de sesión.

2. Conectarse a bases de datos PHP permite interactuar con bases de datos como MySQL, PostgreSQL o SQLite. Puedes guardar, editar, buscar o eliminar datos fácilmente. 

        


<?php
$conn = mysqli_connect("localhost", "usuario", "clave", "basedatos");
?>

3. Generar contenido HTML dinámico En lugar de páginas estáticas, puedes crear sitios que cambien dependiendo del usuario, la hora, el idioma, etc. 

        


<?php
echo "
Bienvenido, $usuario
";

?>

4. Gestionar usuarios y sesiones PHP permite guardar datos del usuario entre páginas usando cookies o sesiones. 

        


<?php
session_start();
$_SESSION["nombre"] = "Carlos";

?>

5. Procesar formularios

PHP recoge la información que el usuario escribe en formularios web y la procesa (como enviar un email, guardar datos, hacer cálculos, etc.).

6. Cargar archivos, imágenes o crear descargas

PHP permite subir archivos al servidor o forzar descargas desde el navegador.

resumen

La función de PHP es convertir una página web en una aplicación interactiva, dinámica y conectada a bases de datos, que responde en tiempo real a lo que hace el usuario.

JavaScript

JavaScript es un lenguaje de programación esencial para la web moderna. Sin él, los sitios web serían simples

documentos estáticos. Gracias a JavaScript, las páginas se vuelven interactivas, dinámicas y capaces de responder a las acciones del usuario en tiempo real.

Aunque fue creado originalmente para ejecutarse en los navegadores, hoy también se usa fuera de ellos (por ejemplo, en servidores, aplicaciones móviles o incluso robots).

Breve historia de JavaScript

1995: JavaScript fue creado por Brendan Eich en tan solo 10 días, mientras trabajaba en Netscape.

Inicialmente se llamó Mocha, luego LiveScript, y finalmente JavaScript (por motivos de marketing, aunque no está directamente relacionado con Java).

En sus inicios fue criticado por ser limitado y poco seguro, pero con el tiempo evolucionó hasta convertirse en un lenguaje robusto y poderoso.

Sintaxis básica de JavaScript

        


// Declarar una variable
let nombre = "Laura";

// Función que saluda al usuario
function saludar(nombre) {
    return "Hola, " + nombre + "!";
}

// Mostrar resultado en consola
console.log(saludar(nombre));

Python

Python es un lenguaje de programación interpretado, de alto nivel, con una sintaxis limpia que se asemeja al inglés. Fue diseñado para ser fácil de leer y escribir, lo que lo convierte en una excelente opción para principiantes. Es multiplataforma y ampliamente utilizado en áreas modernas de desarrollo.


Datos curiosos sobre python:


1-El nombre "Python" no viene de la serpiente, sino del grupo de comedia británico Monty Python’s Flying Circus.

2-Uno de los principios fundamentales de Python es “There should be one—and preferably only one—obvious way to do it”. Esto promueve una forma clara y directa de programar.

3-Python fue adoptado tempranamente por Google como uno de sus lenguajes clave. Incluso su creador, Guido van Rossum, trabajó allí por varios años.


Casos de uso con PYTHON:


CASO: Un trabajador de oficina usa Python para renombrar automáticamente cientos de archivos.
COMO LO HACE: Escribe un codigo que recorre carpetas y cambia nombres según una regla.
BENEFICIO: Ahorra tiempo y reduce errores humanos.


CASO: Quieres que tu computadora te avise a cierta hora algo, por ejemplo, para recordar tomar un descanso o asistir a una clase.
COMO LO HACE: Con unas pocas líneas de código, puedes crear una alarma que espera hasta la hora indicada y luego emite un sonido o muestra un mensaje.
BENEFICIO: Aprendes a trabajar con fechas y horas, bucles while y funciones de tiempo (time.sleep()), mientras haces algo útil para ti..

Ejemplo de codigo para alarma:



import time

hora_objetivo = "15:30"

while True:
   hora_actual = time.strftime("%H:%M")
   if hora_actual == hora_objetivo:
       print("⏰ ¡Es hora de tu descanso!")
       break
   time.sleep(30)

Java

Java es un lenguaje de programación orientado a objetos, compilado a bytecode y ejecutado en la Java Virtual Machine (JVM), lo que lo hace extremadamente portátil. Es más estructurado y requiere más código para tareas simples, pero ofrece robustez y escalabilidad.


Datos curiosos sobre JAVA:


1-Más viejo, pero muy vigente. Java fue lanzado en 1995 y sigue siendo uno de los lenguajes más usados en grandes empresas y sistemas críticos.

2-Java y Minecraft. La primera versión de Minecraft, uno de los videojuegos más vendidos de la historia, fue escrita completamente en Java.

3-Java se llamó originalmente Oak, pero tuvieron que cambiarlo porque ese nombre ya estaba registrado. Eligieron “Java” por el café de la isla de Java.


Casos de uso con JAVA:


CASO: Una persona crea una app móvil de recetas de cocina.
COMO LO HACE: Usa Java en Android Studio para programar la lógica de la app, como mostrar recetas, guardar favoritas, etc.
POR QUE JAVA: Android fue diseñado para funcionar principalmente con Java, lo que lo hace ideal para estas apps.


CASO: Un banco desarrolla su sistema interno de transferencias y gestión de cuentas.
COMO LO HACE: Utiliza Java por su seguridad, rendimiento y estabilidad.
POR QUE JAVA: Es muy confiable en entornos que manejan datos sensibles y deben estar activos 24/7.

Cuadro comparativo (PYTHON vs JAVA)

Caracteristica Python Java
Sintaxis Simple y legible, ideal para escribir rápido Más estricta y detallada, favorece la estructura
Velocidad de ejecución Más lento por ser interpretado Más rápido por ser compilado a bytecode
Curva de aprendizaje Suave y accesible para novatos Requiere más tiempo, pero forma bases sólidas
Portabilidad Alta (corre en múltiples sistemas con pocos cambios) Muy alta (gracias a la JVM "Java Virtual Machine")
Uso movi Limitado: no es común en desarrollo móvil nativo Alto: ampliamente usado en Android
